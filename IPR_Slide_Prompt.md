```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

│          ↓                              │
│ Utilization Rate: 72.7% ✅              │
└─────────────────────────────────────────┘
```

**디자인 포인트**:
- 계산 공식은 배경색이 있는 박스
- 화살표로 흐름 표시
- 결과값(72.7%)은 큰 폰트, 초록색
