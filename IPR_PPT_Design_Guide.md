# IPR Long-term Roadmap PPT - Design Guide

## 📊 **샘플 PPT 분석 결과**

### **1. 색상 팔레트 (Color Scheme)**

#### **Primary Colors**:
```
Navy Blue (Primary):   #232669  RGB(35, 38, 105)
Bright Blue (Accent):  #0583F2  RGB(5, 131, 242)
Light Gray (BG):       #EBECF0  RGB(235, 236, 240)
```

#### **Supporting Colors**:
```
Black (Text):          #000000  RGB(0, 0, 0)
White (BG):            #FFFFFF  RGB(255, 255, 255)
Link Blue:             #0563C1  RGB(5, 99, 193)
```

#### **Color Usage Rules**:
- **제목/헤더**: Navy Blue (#232669)
- **강조/아이콘**: Bright Blue (#0583F2)
- **배경**: White or Light Gray (#EBECF0)
- **본문 텍스트**: Black (#000000)

---

### **2. 타이포그래피 (Typography)**

#### **폰트 패밀리**:
```
Primary (영문):   Montserrat (Sans-serif, Modern)
Primary (한글):   맑은 고딕 (Malgun Gothic)
Secondary:        Montserrat Light (for subtitles)
```

#### **폰트 크기 가이드**:
```
타이틀 슬라이드:
  - Main Title:   32pt (Montserrat Bold)
  - Subtitle:     18-20pt (Montserrat Light)

컨텐츠 슬라이드:
  - Section Title: 28pt (Montserrat Bold, Navy Blue)
  - Body Text:     14-16pt (Montserrat Regular, Black)
  - Caption:       10-12pt (Montserrat Light, Gray)
```

#### **타이틀 슬라이드 (Slide 1)**:
```
┌─────────────────────────────────────────┐
│                                         │
│  [배경 이미지 - 블러 처리]               │
│                                         │
│                                         │
│         Proposal for                    │
│   AI Transformation Organization        │
│         December 2025                   │
│                                         │
│       Presented by James Won            │
│   Chief Business Excellence Officer     │
│         CS WIND Group                   │
│                                         │
│                                         │
└─────────────────────────────────────────┘

Layout:
- 중앙 정렬
- 배경: 어두운 톤 (블러 + 오버레이)
- 텍스트: 흰색 (가독성)
- 층위: 제목 > 부제 > 발표자 정보
```

#### **섹션 타이틀 슬라이드 (Slide 2)**:
```
┌─────────────────────────────────────────┐
│                                         │
│  [좌측 여백]                             │
│                                         │
│        AI eXcellence                    │
│                                         │
│   Driving AI-native transformation      │
│   and operational excellence            │
│   across the global enterprise.         │
│                                         │
│                                         │
└─────────────────────────────────────────┘

Layout:
- 좌측 정렬 (좌측 여백 10-15%)
- 배경: 흰색 또는 Light Gray
- 제목: Large, Bold, Navy Blue
- 본문: Medium, Light, Gray/Black
- 최소주의 (Minimalist)
```

#### **컨텐츠 슬라이드 (Slide 3-8)**:
```
┌─────────────────────────────────────────┐
│  Section Title (상단 좌측, Navy)         │
│  ─────────────                          │
│                                         │
│  • Bullet Point 1                       │
│    - Sub point                          │
│                                         │
│  • Bullet Point 2                       │
│    - Sub point                          │
│                                         │
│  [다이어그램 or 차트]                    │
│                                         │
│  💡 Insight or Key Message (하단)       │
│                                         │
└─────────────────────────────────────────┘

Layout:
- 상단: Section Title + Underline
- 중단: 컨텐츠 (텍스트/다이어그램/차트)
- 하단: Insight Box (배경색 Light Gray)
- 좌우 여백: 10%
- 상하 여백: 8%
```

### **4. 디자인 요소 (Design Elements)**

#### **아이콘 스타일**:
```
Style: Line icons (outline)
Weight: 2-3px
Color: Bright Blue (#0583F2)
Examples:
  🚀 → Rocket (transformation)
  📊 → Chart (analytics)
  ⚡ → Lightning (speed/efficiency)
  🎯 → Target (goal)
  💡 → Light bulb (insight)
```

#### **도형/그래픽**:
```
Shape: Rounded rectangles (corner radius: 10-15px)
Line: 2-3px, Navy Blue or Bright Blue
Shadow: Subtle (opacity 10-20%, offset 3-5px)
Gradient: Avoid (keep flat colors)
```

#### **차트 스타일**:
```
Type: Bar, Line, Donut (simple, clean)
Color: Use palette (Navy, Bright Blue, Light Gray)
Grid: Minimal (light gray, 1px)
Labels: Montserrat, 10-12pt
Legend: Bottom or right
```

---

### **5. 레이아웃 비율 (Proportion Guide)**

#### **슬라이드 크기**:
```
Aspect Ratio: 16:9 (Standard)
Dimensions: 1920 × 1080 px (or 10 × 5.625 in)
```

#### **여백 (Margins)**:
```
Top:    8% of slide height
Bottom: 8% of slide height
Left:   10% of slide width
Right:  10% of slide width

Safe Zone: 80% × 84% (center area)
```

#### **요소 간격 (Spacing)**:
```
Title → Content:  30-40px
Bullet Points:    20-25px (between items)
Paragraph:        15-18px (line height 1.5)
Sections:         50-60px (between major sections)
```

---

### **6. 애니메이션 & 전환 (Animation)**

#### **슬라이드 전환**:
```
Type: Fade (부드럽게)
Duration: 0.5-0.7초
Sound: None
```

#### **오브젝트 애니메이션** (최소 사용):
```
Entrance: Fade In or Wipe
Emphasis: None (avoid)
Exit: None (keep visible)
Timing: 0.3-0.5초 (빠르게)

Rule: 한 슬라이드에 최대 3개 애니메이션
```

---

### **7. 이미지 사용 가이드**

#### **배경 이미지**:
```
Type: 추상적, 기하학적, 기술적
Treatment: Blur (20-30%) + Dark overlay (opacity 40-60%)
Position: Full bleed
Purpose: 타이틀 슬라이드만 사용
```

#### **컨텐츠 이미지**:
```
Type: Icons, diagrams, screenshots
Style: Flat, minimalist
Treatment: Drop shadow (subtle)
Size: 최대 30% of slide width/height
Alignment: Grid-based (align to margins)
```

#### **아이콘 라이브러리 추천**:
```
• Phosphor Icons (https://phosphoricons.com/) - Thin, Modern
• Feather Icons (https://feathericons.com/) - Simple, Clean
• Heroicons (https://heroicons.com/) - Professional
```

---

### **8. 텍스트 계층 구조 (Text Hierarchy)**

#### **우선순위**:
```
Level 1: Section Title
  - Size: 28pt
  - Weight: Bold
  - Color: Navy Blue (#232669)
  - Case: Title Case

Level 2: Sub-heading
  - Size: 20pt
  - Weight: Semi-bold
  - Color: Bright Blue (#0583F2)
  - Case: Sentence case

Level 3: Body Text
  - Size: 14-16pt
  - Weight: Regular
  - Color: Black (#000000)
  - Line Height: 1.5

Level 4: Caption / Note
  - Size: 10-12pt
  - Weight: Light
  - Color: Gray (#666666)
  - Style: Italic (optional)
```

---

### **9. 특수 컴포넌트 (Special Components)**

#### **Insight Box** (💡 Key Takeaway):
```
┌─────────────────────────────────────────┐
│ 💡 Key Insight                          │
│                                         │
│ This is an important message that       │
│ summarizes the main point.              │
└─────────────────────────────────────────┘

Style:
- Background: Light Gray (#EBECF0)
- Border: None or 1px Bright Blue (#0583F2)
- Padding: 20px
- Corner Radius: 10px
- Icon: 💡 (16-20pt)
- Text: 14pt, Regular, Black
```

#### **Stat Card** (숫자 강조):
```
┌─────────────────┐
│      120%       │
│   Efficiency    │
│   Improvement   │
└─────────────────┘

Style:
- Background: White
- Border: 2px Bright Blue (#0583F2)
- Shadow: Drop shadow (subtle)
- Number: 48pt, Bold, Navy Blue
- Label: 12pt, Regular, Gray
- Alignment: Center
```

#### **Timeline / Roadmap**:
```
Q1 ────●──── Q2 ────●──── Q3 ────●──── Q4
       │            │            │
   Foundation   Intelligence  Optimization

Style:
- Line: 2px, Bright Blue
- Dots: 16px circle, Bright Blue (fill)
- Labels: 14pt, Semi-bold, Navy Blue (top)
- Descriptions: 12pt, Regular, Black (bottom)
```

### **10. 슬라이드 템플릿 (Slide Templates)**

#### **Template A: Title Slide**
```
Use: 발표 시작, 섹션 구분
Background: Image (dark) + overlay
Text: Center-aligned, White
Elements: Title + Subtitle + Author
```

#### **Template B: Section Divider**
```
Use: 큰 섹션 전환 (Phase 1, 2, 3)
Background: Solid (Navy Blue or Bright Blue)
Text: Large, Bold, White
Elements: Section number + Title
```

#### **Template C: Content (Text Heavy)**
```
Use: 설명, 개념 정의
Layout: Title (top) + Bullets (center)
Background: White or Light Gray
Text: Left-aligned, Hierarchical
```

#### **Template D: Content (Visual Heavy)**
```
Use: 다이어그램, 차트, 프로세스
Layout: Title (top) + Visual (center) + Caption (bottom)
Background: White
Elements: Diagram/Chart + Insight Box
```

#### **Template E: Comparison**
```
Use: Before/After, Options A vs B
Layout: Title (top) + 2-3 columns (center)
Background: White
Elements: Side-by-side boxes or tables
```
